Structs

Recursion

  • Example: Factorial

  • Base Case / Recursive Case

  • Call Stack

  • Multiple Base Cases: The Fibonacci Sequence 斐波那契数列

  • Multiple Recursive Cases: The Collatz Conjecture 科拉茨猜想

int collatz(int n)
{
// base case
if (n == 1)
    return 0;
// even numbers
else if ((n % 2) == 0)
    return 1 + collatz(n/2);
// odd numbers
else
    return 1 + collatz(3*n + 1);
}
